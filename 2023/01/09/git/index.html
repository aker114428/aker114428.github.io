通过多个ssh密钥部署多个账户的仓库

一个公钥只能配置一个账户，一个账户可以关联多个公钥。比如你拥有多个设备，每个设备上可以生成一个公钥和你的账户相关联，关联后此公钥便不能与其它的账户或项目进行关联.否则会出现报错
